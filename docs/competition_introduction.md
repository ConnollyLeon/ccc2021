# 竞赛介绍

CCF TCARCH- 定制计算算法实现挑战赛是中国计算机学会体系结构专委会所举办的旨在挖掘计算机体系结构人才、激发学生对计算机体系结构的研究兴趣、培养学生创新精神的一项全国赛事

# 竞赛日程
